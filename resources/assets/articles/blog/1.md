### Part-I 安装Nginx ####
1. 安装PCRE
2. 下载 ngx_cache_purge 并解压，用来清除缓存
3. 下载Nginx并解压
4. cd nginx-1.7.7
5.  编译，--prefix使用默认值，则nginx安装在/usr/local/nginx
```bash
./configure    --user=www    --group=www    --add-module=../ngx_cache_purge-1.0 
    --with-http_stub_status_module    --with-http_ssl_module
make && make install
```
